Core Readers

This section describes Pathom built-in readers provided in the Pathom core namespace.

map-reader

Map reader get the :key from the current node and look it up in the ::p/entity map.

It runs with the following behavior:

  • If the attribute requested exists in the current parsing context (with any value, even nil), it returns it.

  • If the attribute is missing, it returns ::p/continue, which is an indication to move to the next reader in the chain.
