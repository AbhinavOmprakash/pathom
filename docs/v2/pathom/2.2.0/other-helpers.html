p/map-select

Imagine select-keys on steroids! This helper function takes a nested map and filters its contents according to an EQL selection. Example:

(p/map-select {:foo "bar" :deep {:a 1 :b 2}} [{:deep [:a]}])
=> {:deep {:a 1}}
